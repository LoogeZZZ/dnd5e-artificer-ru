# Изобретатель (Artificer) — RU

Модуль для **Foundry VTT v13** (система **D&D 5e**, dnd5e ≥ 4.0.0), добавляющий класс
**Изобретатель (Artificer)** из *Tasha's Cauldron of Everything* полностью на русском языке.

## Что внутри

Компендиум **«Изобретатель (Artificer)»** (`izobretatel`), сгруппированный по папкам:

- **Класс** — Изобретатель (1–20 ур.) с полной цепочкой развития (advancement):
  кость хитов, спасброски, владения, выбор навыков, увеличения характеристик,
  автоматическая выдача классовых умений по уровням, выбор инфузий и выбор подкласса на 3 ур.
- **Подклассы** (специальности): **Алхимик**, **Артиллерист**, **Боевой кузнец**, **Бронник** —
  каждый с автоматической выдачей своих умений на 3/5/9/15 ур.
- **Классовые умения** (11): Магический мастеровой, Использование заклинаний, Инфузирование
  предмета, Подходящий инструмент, Компетентность во владении инструментами, Проблеск
  гениальности, Эксперт/Учёный/Мастер в обращении с магическими предметами, Хранящий
  заклинания предмет, Душа изобретения.
- **Умения подклассов** (26) — все умения четырёх специальностей.
- **Инфузии** (16) — весь список инфузий изобретателя с описаниями.

Всего **58 предметов** + 5 папок.

## Установка

1. Скопируйте папку `dnd5e-artificer-ru` целиком в каталог данных Foundry:
   `…/FoundryVTT/Data/modules/dnd5e-artificer-ru`
   (внутри должны быть `module.json`, `packs/`, `lang/`).
2. Запустите мир на системе **dnd5e**, в **Управление модулями** включите
   «Изобретатель (Artificer) — RU».
3. Откройте боковую вкладку **Компендиумы → Изобретатель (Artificer)**.

## Как пользоваться

1. Перетащите предмет **Изобретатель** (папка «Класс») на лист персонажа.
   Система проведёт по шагам развития: кость хитов, владения, навыки, **выбор 4 инфузий**.
2. На **3-м уровне** появится выбор **подкласса** — перетащите нужный подкласс
   из папки «Подклассы» на лист (или выберите через шаг развития класса).
3. Классовые и подклассовые умения, а также увеличения характеристик и новые инфузии
   выдаются автоматически при повышении уровня.

## Важные примечания

- **Заклинания.** Класс настроен как «полукастер» (прогрессия `artificer`, базовая
  характеристика — Интеллект). Сами заклинания берутся из стандартного компендиума
  заклинаний системы dnd5e (SRD). «Всегда подготовленные» заклинания подклассов
  перечислены в описаниях соответствующих умений (для удобного добавления вручную).
- **Спутники** (Стальной защитник, Гомункул, Мистическая пушка) и предметы инфузий
  оформлены как описательные умения; при необходимости актёров-спутников создавайте
  отдельно — механика подробно описана в тексте умений.
- Терминология выверена по русскому сообществу D&D (dnd.su).

## Пересборка компендиума

Контент хранится в читаемом виде в `content.mjs`, исходные документы — в `src/`.
Для пересборки LevelDB-пака:

```bash
npm install
node build.mjs    # пересобирает packs/izobretatel и src/
node verify.mjs   # проверяет целостность ссылок развития
```
